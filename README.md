# Skydrill

Distribution of Apache Presto for Microsoft Windows.

### Features

- Coordinator High Availability (HA)
- Azure Blob and Data Lake Connectors
- Sharded SQL (JDBC) Connector
- Dynamic Catalog APIs
- View Push-down (JDBC and Thrift)
- Windows Hosting
